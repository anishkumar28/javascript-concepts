
 Call Stack is a stack, it contains the execution context in its bottom, everytime when a new function is invoked the execution context is pushed in the stack and when it's completed it got popped out.


 " Call stack maintains the order of execution of execution context. "


There are many names of Call Stack in Javasript"-

 1. Call Stack
 2. Execution Context Stack
 3. Program Stack
 4. Control Stack
 5. Runtime Stack
 6. Machine Stack
